# AegisStadium AI - Knowledge Base

This directory contains stadium layouts, evacuation guides, emergency protocols, and accessibility rules used by the AI RAG Engine.

## Subdirectories
- `stadium-guides/`: Seating maps, entry gate mappings, facility guides.
- `evacuation/`: Fire escape, evacuation pathways, shelter zones.
- `fifa/`: Match operations rules, VIP protocols, media guidelines.
- `accessibility/`: Wheelchair pathways, elevator listings, audio navigation aids.
- `transportation/`: Transit schedules, shuttle route data, smart parking areas.
- `embeddings/`: Cached text vectors.
